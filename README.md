## <img src="https://user-images.githubusercontent.com/74038190/212284100-561aa473-3905-4a80-b561-0d28506553ee.gif" width="28"> About This Project

This repository contains the complete, production-ready source code for my personal portfolio website. It is designed to showcase my skills, projects, and professional journey as a **Web Developer**, **Indie Game Dev**, and **AI Automation Builder**.

The site is hosted directly via **GitHub Pages** and is optimized for performance, accessibility, and a premium user experience across all devices.

<br/>

## <img src="https://user-images.githubusercontent.com/74038190/212284136-0398950e-4c9f-4c99-b5b0-c3b3d5c9b1d0.gif" width="28"> ✨ Key Features

<div align="center">

| Feature | Description |
| :--- | :--- |
| 🌓 **Dynamic Theming** | Seamless Light/Dark mode toggle with persistent `localStorage` memory. |
| 📱 **Fully Responsive** | Flawless adaptation across mobile, tablet, and desktop viewports. |
| 🎨 **Modern Aesthetics** | Clean monochrome design, subtle noise textures, grid patterns, and glassmorphism. |
| ⚡ **Smooth Animations** | Scroll-triggered reveal animations powered by the AOS (Animate On Scroll) library. |
| 🖼️ **Custom SVG Art** | Hand-coded, theme-aware SVG illustrations for projects and the "About" section. |
| 🚀 **Zero Build Step** | Pure HTML, CSS (Tailwind via CDN), and Vanilla JS for instant loading and easy maintenance. |

</div>
